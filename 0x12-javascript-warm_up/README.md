# JavaScript - Warm up


Project done during **Full Stack Software Engineering studies** at **Holberton School**. It aims to learn about docstrings to create and implement tests in **Python**.

## Technologies
* Your code should be semistandard compliant (version 16.x.x). Rules of Standard + semicolons on top. Also as reference: AirBnB style
* All your files will be interpreted on Ubuntu 20.04 LTS using node (version 14.x)
* Tested on Ubuntu 14.04 LTS

## Files

| Filename | Description |
| -------- | ----------- |
| `0-javascript_is_amazing.js` | Prints “JavaScript is amazing” using a constant variable |
| `1-multi_languages.js` | Prints 3 lines: “C is fun”, “Python is cool”, “JavaScript is amazing” |
| `2-arguments.js` | Prints a message depending on the number of arguments passed: “No argument”, “Argument found”, or “Arguments found” |
| `3-value_argument.js` | Prints the first argument passed, or “No argument” if none |
| `4-concat.js` | Prints two arguments passed in the format: `<arg1> is <arg2>` |
| `5-to_integer.js` | Prints “My number: <int>” if the first argument can be converted to an integer, otherwise prints “Not a number” |
| `6-multi_languages_loop.js` | Prints 3 lines using an array and a loop: “C is fun”, “Python is cool”, “JavaScript is amazing” |
| `7-multi_c.js` | Prints x times “C is fun”, where x is the first argument. Prints “Missing number of occurrences” if the argument is missing or invalid |
| `8-square.js` | Prints a square of size given by the first argument using the character X. Prints “Missing size” if the argument is missing or invalid |
| `9-add.js` | Prints the addition of 2 integers passed as arguments using a function `add(a, b)` |
| `10-factorial.js` | Computes and prints the factorial of the first argument recursively. Prints 1 if no argument or invalid argument is given |
| `11-second_biggest.js` | Prints the second biggest integer from the list of arguments. Prints 0 if none or only one argument is passed |
| `12-object.js` | Updates the value of `myObject.value` from 12 to 89 and prints the object before and after the update |
| `13-add.js` | Defines and exports a function `add(a, b)` that returns the addition of 2 integers |
